# Privacy Notes

## Dados coletados no app publico
- sintomas marcados pelo usuario
- localizacao aproximada para enquadramento territorial
- identificador tecnico do aparelho para limitar repeticoes
- endereco de rede para protecao antifraude

## Finalidade
- compor o radar epidemiologico publico
- reduzir envio duplicado ou abusivo
- manter a confiabilidade da leitura territorial

## O que o app nao faz
- nao entrega diagnostico medico
- nao expõe a camada institucional do SaaS
- nao libera envio irrestrito da mesma rede ou aparelho

## Mensagem sugerida para politica publica
Os dados enviados pelo aplicativo sao tratados para fins de monitoramento epidemiologico territorial, seguranca antifraude e melhoria da qualidade do monitoramento publico.
