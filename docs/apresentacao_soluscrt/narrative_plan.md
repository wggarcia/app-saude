# Plano narrativo - Apresentacao SolusCRT Saude

Publico: gestores publicos, empresas, hospitais, farmacias, investidores e parceiros.
Objetivo: explicar o problema epidemiologico, posicionar o SolusCRT como sala de controle com IA e conduzir para reuniao de diagnostico/contrato.
Arco: urgencia do problema -> solucao integrada -> valor por setor -> seguranca -> modelo comercial -> chamada para acao.
Sistema visual: fundo escuro institucional, azul petroleo, verde ciano, amarelo alerta e vermelho risco; cards de controle, mapas abstratos e linhas de fluxo.
Editabilidade: todos os textos, cards, numeros e tabelas como objetos editaveis do PowerPoint.
Slides: abertura, problema, solucao, fluxo, mapa, fontes oficiais, empresas, farmacias, hospitais, governo, seguranca, precificacao, frase de impacto, chamada final.
