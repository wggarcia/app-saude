## Regra de governanca
1. Toda mudanca de texto contratual deve atualizar este registro.
2. Mudancas juridicas relevantes devem subir versao **minor** (ex: 1.1.0) ou **major** (ex: 2.0.0).
3. Correcao textual sem impacto juridico pode subir versao **patch** (ex: 1.0.1).
4. Nenhuma versao anterior deve ser apagada do historico.
